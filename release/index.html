macOS

Install on macOS

  1. Download the DMG and open it.
  2. Drag Strata to Applications.
  3. Open Strata from Applications.
  4. If macOS blocks it: System Settings → Privacy & Security → “Open Anyway”.
  5. If it’s still blocked, run this in Terminal:
sudo xattr -r -d com.apple.quarantine "/Applications/Strata.app"

Windows

Install on Windows

  1. Download the installer (.exe).
  2. Run it.
  3. If SmartScreen appears: “More info” → “Run anyway”.

Feedback

If you run into bugs, type bug into the Strata pill to submit a bug report for future development. Open the pill with Cmd/Ctrl + Shift + Space, then type tutorial for the basics.
